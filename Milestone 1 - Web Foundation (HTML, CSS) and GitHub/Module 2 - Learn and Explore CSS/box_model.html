CSS Box Model

This box demonstrates the CSS Box Model.

The box model consists of: content, padding, border, and margin.

Total Count

Width: 200px (content) + 40px (padding: 20px on each side) + 10px (border: 5px on each side) + 30px (margin: 15px on each side) = 280px
Height: 100px (content) + 40px (padding: 20px on top and bottom) + 10px (border: 5px on top and bottom) + 30px (margin: 15px on top and bottom) = 180px
